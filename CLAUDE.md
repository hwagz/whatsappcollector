Project: WhatsApp Group Message Collector

Goal:
Build a small service that connects to WhatsApp Web and records group messages into a SQLite database.

Tech stack:
- Node.js
- whatsapp-web.js
- SQLite

Project constraints:
- Only store group messages
- Ignore private messages
- Store message metadata and body
- No AI features yet

Development workflow:
- Plan before writing code
- Keep the project simple
- Explain architecture decisions
- Show full updated files when writing code
